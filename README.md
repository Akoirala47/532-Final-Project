# PySpark Performance Benchmarking Study

**Team:** Aayush Koirala, Benson Zheng  
**Course:** COMPSCI 532 — Systems for Data Science (SP26)

---

## Overview

This project extends our PySpark text analysis homework into a full-scale benchmarking study. We scale word count, word frequency, and word pair analyses to a large multi-document corpus, rewrite queries in **Spark SQL**, and measure performance across all approaches.
